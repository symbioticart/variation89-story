# Variation 89 — Story

Vertical (1080×1920) Instagram-Stories export of the symbiotic painting
*Variation 89* by Nikolai Grigoriev — a generative oil driven by the artist's
own Oura Ring biometrics, presented as a hung object on a wall with a museum
plate.

- One portrait painting per day, deterministic from the day's body-state.
- Opens on the present day; `?day=N` renders any day in the snapshot; `?dead`
  shows the silence/death state.
- Tap the **top-left corner** of the screen to download the full-resolution
  1080×1920 PNG to the device.

Zero dependencies. `npm start` → static server on `PORT` (default 3470).
